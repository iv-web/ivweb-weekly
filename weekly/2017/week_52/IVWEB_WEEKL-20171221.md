<h1 id="#title_26" >27、因违反软件商店政策 微软下架谷歌的 Chrome 安装程序应用</h1>

[https://www.oschina.net/news/91700/microsoft-google-chrome-windows-store-removal](https://www.oschina.net/news/91700/microsoft-google-chrome-windows-store-removal)
<p>谷歌在微软的 Windows Store 上传了一个 Chrome 的安装程序应用，它会引导 Windows 10 用户下载安装 Chrome 浏览器。当然，Chrome 浏览器本身并没在微软的 Windows Store 发布。</p><p><img src="https://static.oschina.net/uploads/space/2017/1220/145233_n3PN_2720166.jpg"/><em><br/>被微软下架的应用</em></p><p>微软随后以违反应用商店政策为由将谷歌上传的应用下架。微软方面表示，为了确保应用能提供“独特”的价值，它欢迎谷歌在遵守商店政策的条件下构建 Microsoft Store 浏览器应用。<br/></p><p>看起来谷歌不会愿意接受这个邀请。谷歌不愿意将 Chrome 带入 Windows Store 有很多原因，但主要原因可能与 Windows 10 S 的限制有关。Windows 10 S 限制非应用商店之外的应用下载和安装。事实上，大多数 Windows 10 机器都没安装 Windows 10 S，它的使用率并不高，所以谷歌没有什么动机去开发一个专门的版本以满足 Windows Store 的要求。<br/></p><p>我们甚至可以认为，谷歌在 Windows Store 上传&nbsp;Chrome 的安装程序应用可能就是为了应对微软 Windows 10 S 操作系统的限制，毕竟该系统中的 Windows Store 要求所有应用都必须使用 Edge 的渲染引擎，而 Chrome 的渲染引擎是 Blink。</p><p>据外媒消息，对于微软下架谷歌的 Chrome 安装程序应用这件事，谷歌方面暂未发表评论。</p><p>来自：</p>
