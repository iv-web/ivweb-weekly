<h1 id="#title_20" >21、开发者社区 Stack Overflow 大规模裁员 占比 20%</h1>

[https://www.oschina.net/news/90250/stack-overflow-lays-off-staff](https://www.oschina.net/news/90250/stack-overflow-lays-off-staff)
<p><img src="https://static.oschina.net/uploads/space/2017/1104/082640_P3qG_2720166.png"/></p><p>裁员年年都有，但 2017 年似乎是 IT 界最为动荡的一年。从微软、Oracle、IBM，到思科、思杰、HPE，再到 Etsy、Facebook、Slack，无论是处于转型变革中的老牌巨头，还是日渐成熟的创新型公司，在动荡的科技行业，裁员风波一浪高过一浪，似乎并没有消停下来的意思。<br/></p><p>据&nbsp;透露，11 月 3 日，全球著名技术问答社区 Stack Overflow 正在进行大规模裁员，涉及其 300 名员工中的将近 20%。</p><p>经 TechCrunch 了解和证实，运营 Stack Overflow 社区的公司将把工作重点转向核心的问答产品，剔除需求较少的业务，例如销售和营销岗位将会进行重组。作为这一举措的一部分，该公司还关闭了在丹佛的办事处。</p><p>该公司的一份声明指出：“Stack Overflow 决定将产品开发扩展到我们的核心问答产品中，包括 Stack Overflow 企业和开发人员合作频道，以及众所周知的开发人员知识共享核心平台。这些都是准备充分且客户需求强烈的解决方案。所以很不幸地，我们对一些销售和营销岗位进行了重组，包括关闭丹佛办事处和削减部分员工。Stack Overflow 的客户和忠诚用户将会看到我们的服务没有中断。所有产品的业务都在增长，这一举措甚至会在未来几个月加速使得更多功能和产品走向市场。”<br/></p><p>消息人士还表示，该公司正在寻求关闭 Careers 功能，但 Stack Overflow 的“开发人员岗位”功能仍继续存在。事实上，在公司提供的声明中，并没有提到 Stack Overflow 的 Careers 业务。</p><p>这里颇值得玩味，毕竟这与 Stack Overflow 以往所表现出来的乐观情绪相去甚远，2015 年该公司融资 4000 万美元时曾表示，将专门花力气建立该项业务的招聘环节，这部分在当时占全公司收入的三分之二，在 Stack Overflow 建立声誉和问答流量增长中居功至伟。</p><p>作为开发者心中的圣地，包括 Stack Overflow 在内，HackerNews、GitHub 上同样汇聚了一批优秀的开发者。面对潜在的竞争对手，Stack Overflow 首席执行官和联合创始人 Joel Spolsky 表示，“开发者使用我们有其他的需求，我们这几个社区的业务并没有真正重叠。他们都在开发者关注的范围内，但还有很多其他的东西，比如魔兽世界。我们在问答领域的主导地位，等同于 GitHub 在社交化编码和版本控制方面的影响力。”<br/></p><p>目前还不清楚 Stack Overflow 的主要营收来自何处，但该公司似乎正在把更多赌注押到付费的企业服务上。<br/></p><p>频道（Channels）功能是一款仍在测试阶段的新产品，属于私人版本的Stack Overflow，只在企业团队之间使用。相对于该公司的开放式问答业务，这是一个有趣的转折点，有可能使它进入一个已经具有竞争力的领域。<br/></p><p>在当前的企业级服务市场，Slack、HipChat 等企业内部聊天软件人气十足，微软也在去年推出了同类产品 Teams 以抗衡 Slack 的潜在竞争。</p><p>如果这个模式可复制，那么 Stack Overflow 这个基于开放式的问答社区，尝试&nbsp;Channels 这样的企业级业务是否能找到一条盈利的通道？<br/></p><p>再看回今年的裁员风波，背后是否又暗藏着什么危机？事实上，市场环境、战略转型求生存、员工个人竞争力不足，这些往往都会成为企业裁员的因素之一。</p><p>此外，近两年人力成本居高不下，税、费等政策环境的日益恶劣，精明的投资者们也开始权衡风险，那些不惜成本求增长和高烧钱率的公司将被市场抛弃，而只有那些真正有竞争力的公司，有良好的商业模式和战略性的营销方式才可成为市场的宠儿。</p><p>可能有人会说，如果有一天 Stack Overflow 无法访问，这将是码农们的末日。这一天会到来吗？</p><p>到目前为止，Stack Overflow 获得的融资约达 6800 万美元，投资者包括安德森·霍洛维茨（Andreessen Horowitz）、联合广场创投，星火资本（Spark Capital）、指数公司（Index Ventures）、贝佐斯探险计划、创始人集体和一些个人投资者。</p>
