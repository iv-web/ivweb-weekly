<p>开发人员经常使用基于 Linux 的操作系统来完成日常工作和开发新东西。 他们选择 Linux 发行版进行编程的主要考虑因素是兼容性，功耗，稳定性和灵活性。 像 Ubuntu 和 Debian 这样的发行版已经成为了首选。 其他一些很好的选择如 openSUSE，Arch Linux 等。而对于打算购买 Raspberry Pi 的用户，Raspbian 就是最好的选择。</p><p>国外技术网站  近日梳理了 2018 年最值得开发者尝试 9 个发行版，内容仅代表作者个人观点。读者如有不同意见，欢迎留言分享。</p><h3>1. Debian GNU/Linux</h3><p><img height="359" data-cke-saved-src="https://static.oschina.net/uploads/space/2018/0112/183619_PVYA_2903254.png" src="https://static.oschina.net/uploads/space/2018/0112/183619_PVYA_2903254.png" width="638"/></p><p><strong>上榜理由</strong>：Debian 含有大量的软件包，提供良好的稳定性和大量的教程，帮助开发人员解决问题。Debian 测试分支，它有所有最新的软件，并且非常稳定。适合高级程序员和系统管理员。Debian 有很多开源库，另外，它的 .deb 软件包管理也是值得推荐的一点。</p><h3>2. Ubuntu</h3><p><img data-cke-saved-src="https://static.oschina.net/uploads/space/2018/0101/074601_Uuam_2896879.jpg" src="https://static.oschina.net/uploads/space/2018/0101/074601_Uuam_2896879.jpg"/></p><p><strong>上榜理由</strong>：多年来，得益于 Canonical 和开源社区的支持，Ubuntu 已经成为 Linux 桌面场景的一大亮点。由于其受欢迎程度和持续发展，这个 Debian 衍生物在云和服务器应用程序中被大量使用。它也有多种风格来满足人们的不同需求。LTS 版本提供了良好的体验，可以快速解决问题。 Ubuntu 也支持流行的 .deb 包管理系统。</p><h3>3. openSUSE</h3><p><img height="360" data-cke-saved-src="https://static.oschina.net/uploads/space/2018/0112/183935_i6Qu_2903254.png" src="https://static.oschina.net/uploads/space/2018/0112/183935_i6Qu_2903254.png" width="641"/></p><p><strong>上榜理由</strong>：openSUSE 是一个非常稳定的编程操作系统，它可以轻松地让 Ubuntu 运行起来。 这个 Linux 发行版有两个版本。openSUSE Leap 是 LTS 版本，保持更新的同时保证稳定性，Tumbleweed 是为那些喜欢最新软件的用户准备的滚动版本。OpenSUSE 的主要优势之一就是它的 YaST 包管理，可以轻松实现不同的任务自动化。 还有一个就是它的软件交付方法。&nbsp;</p><h3>4. Fedora</h3><p><img data-cke-saved-src="https://static.oschina.net/uploads/space/2017/1130/163919_bUcx_2896879.png" src="https://static.oschina.net/uploads/space/2017/1130/163919_bUcx_2896879.png"/></p><p><strong>上榜理由</strong>：Fedora 的赞助商是红帽公司，以提供 Linux 桌面世界最尖端的功能而闻名。其最新版本经常激发其他 Linux 发行版采用新功能并进行更改。其智能的自动配置和更新的软件包使其成为开发人员完美的编程操作系统。Fedora 只包含开源组件，适合开源爱好者。即使貌似 Linus Torvalds 就很喜欢 Fedora。&nbsp;它在大多数硬件上都是开箱即用的。小版本也提供了尖端的功能。</p><h3>5. CentOS</h3><p><img height="379" data-cke-saved-src="https://static.oschina.net/uploads/space/2018/0112/183714_VRe3_2903254.png" src="https://static.oschina.net/uploads/space/2018/0112/183714_VRe3_2903254.png" width="641"/></p><p><strong>上榜理由</strong>：由于它是从RHEL源代码编译的，所以为 RHEL 构建的大多数商业软件都可以在 CentOS 上运行。它的安装和设置过程几乎就像 Fedora 一样。 它大量的红帽软件集合和 CentOS 存储库能满足不同的软件需求。 它允许使用 Xen 虚拟化来开发应用程序。 它使用 YUM 进行包管理。</p><h3>6. Gentoo</h3><p><img height="392" data-cke-saved-src="https://static.oschina.net/uploads/space/2018/0112/191542_x80t_2903254.png" src="https://static.oschina.net/uploads/space/2018/0112/191542_x80t_2903254.png" width="700"/></p><p><strong>上榜理由</strong>：它可以完成源代码的所有工作，还可以为特定的 CPU 架构重建整个系统。它对技术要求较高，但是时间越久，钻得越深，懂得越多。Gentoo 安装时候有些折腾，但你得到的是一个超级稳定的系统。</p><h3>8. Manjaro Linux</h3><p><img height="360" data-cke-saved-src="https://static.oschina.net/uploads/space/2018/0112/183740_Gc9Z_2903254.png" src="https://static.oschina.net/uploads/space/2018/0112/183740_Gc9Z_2903254.png" width="639"/></p><p><strong>上榜理由</strong>：如果您知道如何设置 Arch Linux 系统，则不需要 Manjaro Linux。但是，如果你想要一个基于 Arch 的系统，而不需要考虑更新，你可以试试 Manjaro。安装非常简单，就像 Ubuntu 或 Linux Mint 一样。 Manjaro 是这个名单上最新的发行版，并且正在快速增长。目前，在 Distrowatch 过去 6 个月的排名中位列第三。</p><h3>9. Raspbian</h3><p><img height="379" data-cke-saved-src="https://static.oschina.net/uploads/space/2018/0112/191135_8VS7_2903254.png" src="https://static.oschina.net/uploads/space/2018/0112/191135_8VS7_2903254.png" width="622"/></p><p><strong>上榜理由</strong>：每一次新的迭代，都使得树莓派微型计算机越来越强大。 它被广泛用于 Linux 编程和编码基础知识的教学。&nbsp;它有 BlueJ，Geany，Greenfoot，Mathematica，Python，Node-RED，Scratch 和其他工具来提高学习效率。&nbsp;</p>
